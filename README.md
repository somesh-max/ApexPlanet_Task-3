# 📊 ApexPlanet Data Analytics Internship – Task 3

## 📌 Project Title

**Data Visualization & Interactive Dashboard using Python and Power BI**

---

## 📖 Overview

This project is completed as part of the **ApexPlanet Software Pvt. Ltd. Data Analytics Internship – Task 3**.

The objective of this task is to perform data visualization using Python libraries and build an interactive dashboard in Power BI to analyze supermarket sales data and derive meaningful business insights.

---

## 🎯 Objectives

- Perform Exploratory Data Visualization.
- Create insightful charts using Python.
- Build an interactive dashboard in Power BI.
- Analyze sales trends and customer behavior.
- Present business insights through visualizations.

---

## 📂 Dataset

**Dataset Name:** SuperMarket Analysis

The dataset contains supermarket transaction records, including:

- Invoice ID
- Branch
- City
- Customer Type
- Gender
- Product Line
- Unit Price
- Quantity
- Tax (5%)
- Sales
- Date
- Time
- Payment Method
- Cost of Goods Sold (COGS)
- Gross Income
- Gross Margin Percentage
- Rating

## 📌 Key Insights

- Yangon generated the highest overall sales.
- Food and Beverages was one of the top-performing product lines.
- E-wallet and Cash were the most frequently used payment methods.
- Female customers contributed slightly more transactions than male customers.
- Sales remained consistent across different branches with minor variations.

## ▶️ How to Run

1. Clone this repository.
2. Install the required Python libraries.

```bash
pip install pandas numpy matplotlib seaborn
```

3. Open **task3_visualization.ipynb** in Jupyter Notebook or VS Code.
4. Run all notebook cells.
5. Open **supermarket_dashboard.pbix** in Power BI Desktop to explore the interactive dashboard.
